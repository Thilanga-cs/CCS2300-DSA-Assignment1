# Module 1 – Smart City Route Planner

## 📌 Overview
This module implements a Smart City Route Planner using fundamental data structures.

It demonstrates:
- Binary Search Tree (BST) for storing locations
- Graph (Adjacency List) for representing roads
- BFS (Queue) traversal
- DFS (Stack) traversal
- Menu-driven console interface

---

## 🧠 Data Structures Used

### 1️⃣ Binary Search Tree (BST)
- Stores city locations in sorted order
- Prevents duplicate locations
- Supports insertion and deletion

### 2️⃣ Graph (Adjacency List)
- Represents roads between locations
- Undirected graph
- Efficient neighbor storage

### 3️⃣ Traversal Algorithms
- **BFS (Breadth First Search)** using Queue
- **DFS (Depth First Search)** using Stack

---

## ⚙ Functional Features

- Add location
- Remove location
- Add road between locations
- Remove road
- Display all connections
- Traverse using BFS
- Traverse using DFS
- Display locations in sorted order (BST)
